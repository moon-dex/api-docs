    Rest API

    Introduction

    These API calls do not require any user authentication. Generally these calls provide publicly available information about the state of markets.

    List Token

    Return all active Assets.

    {
    "code": 0,
    "data": [
        {
            "currency_id": 4,
            "short_name": "FC",
            "name": "FC",

    HTTP Request

    GET https://api.moondex.io/opt/trade/currencies

    Query Parameters

    Name Type Check Description Example
    start int page start from 0 start
    limit int page size default 10

    Response Parameters

    Name Type Check Description Example
    currency_id int asset id 1
    short_name string asset short name FC
    name string asset name FC
    price_usd float asset usd price 0.15
    price_cny float asset cny price 1.05

    Get a Token

    Return a token info by asset query parameter;

    {
    "code": 0,
    "data": [
        {
            "currency_id": 4,
            "short_name": "FC",
            "name": "FC",
            "price_usd": "0.1543",
            "price_cny": "1.05934"
        }
    ]
}

    HTTP Request

    GET https://api.moondex.io/opt/trade/currencies?asset=FC

    Query Parameters

    Name Type Check Description Example
    asset string Token Asset Name FC、BTC、ETH

    Response Parameters

    Name Type Check Description Example
    currency_id int asset id 1
    short_name string asset short name FC
    name string asset name FC
    price_usd float asset usd price 0.15
    price_cny float asset cny price 1.05

    HTTP Request

    GET https://api.moondex.io/opt/trade/pairs

    Query Parameters

    Name Type Check Description Example
    start int page start from 0 start
    limit int page size default 10

    Response Parameters

    Name Type Check Description Example
    pair_id int pair id 1
    alias_name string alias name BTC/USDT
    pair_name string pair name BTC-USDT
    open_price float pair 24h open price 11164.0600000
    close_price float pair 24h close price 11564.0600000
    high_price float pair 24h high price 11864.0600000
    low_price float pair 24h low price 11064.0600000
    price float pair lastest price 11564.025
    baseAssetVolume24h float Total volume of base currency 24h 22
    quoteAssetVolume24h float Total volume of quote currency 24h 3212222
